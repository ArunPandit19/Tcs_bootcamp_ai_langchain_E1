This project implements an AI‑powered customer support agent using:
- Groq LLMs (Mixtral / Llama3)
- LangChain
- LangGraph
- Tool‑Calling Agents
- Structured workflow nodes
The system processes customer emails and performs:
- Classification (urgency + topic)
- Tool‑calling agent reasoning
- Decision making
- Follow‑up planning
- Final structured output
